# Automated-HR-Headcount-Reconciliation-Attrition-Dashboard
I transformed a labor-intensive, manual Excel process into a fully automated Power BI solution for HR leadership. Previously, the team manually reconciled "People by Assignment" (PBA) data across regional locations to identify monthly discrepancies—a process prone to human error and significant delays.
## The Solution:

Multi-Source Integration: Architected a robust data model connecting three distinct data streams: PBA (Active Headcount), Separations, and New Joiners.

Complex Logic & DAX: Developed advanced measures to calculate critical KPIs, including Rolling Attrition Count, In-Period Separations, and complex PBA movement logic.

End-to-End Automation: Built an automated ETL pipeline that eliminated manual calculations, providing a "single source of truth" with monthly scheduled refreshes.

## The Impact:

Efficiency: Reduced manual reporting time by 28 hours per month.

Accuracy: Eliminated regional data discrepancies through standardized logic.

NOTE: Due to strict data privacy and corporate compliance policies, specific dashboard visuals are not displayed to protect sensitive HR information.
